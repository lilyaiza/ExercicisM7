1. Per poder realitzar aquest exercici es necessari descarregar aquest gz, el qual conté la BDD "World": 

http://downloads.mysql.com/docs/world.sql.gz

2. Descomprimim el world.sql.gz

3. Fem la següent comanda per insertar la BDD dins de MYSQL:

"sudo mysql -u [NOM DE USUARI] -p world < world.sql"

4. Anem a la terminal i posem la següent comanda per poder treballar amb MYSQL i PHP:

sudo apt install php-mysql

5. A partir d'aqui ja podriem treballar amb la nostre BDD World al nostre MYSQL.
